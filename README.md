## Usage

This crate is [on crates.io](https://crates.io/crates/tokio-i2c) and can be
used by adding `tokio-i2c` to the dependencies in your project's `Cargo.toml`:

```toml
[dependencies]
tokio-i2c = "0.1"
```

and this to your crate root:

```rust
extern crate tokio_i2c;
```
